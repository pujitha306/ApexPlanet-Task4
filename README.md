# ApexPlanet Task 4 - Sales Dashboard Analysis

## Objective
The objective of this project is to analyze sales data using Microsoft Power BI, create an interactive dashboard, generate business insights, and validate findings using hypothesis testing.

## Tools Used
- Microsoft Power BI
- Microsoft Excel

## Dataset
- Dataset Name: Sales Dataset
- Total Records: 1000
- Total Columns: 10

## Dashboard Visuals
- Total Sales Card
- Sales by Category
- Sales by City
- Orders by Product
- Orders by Gender

## Key Findings
- Total Sales: 139.40M
- Electronics category generated the highest sales.
- Patna recorded the highest sales among all cities.
- Mobile products received the highest number of orders.
- Male and Female orders are nearly equal.

## Hypothesis Testing
- Test Used: Independent Sample T-Test
- Confidence Level: 95%
- P-Value: 0.32
- Result: Since p-value > 0.05, we fail to reject the Null Hypothesis.
- Conclusion: Gender has no significant impact on Total Sales.
